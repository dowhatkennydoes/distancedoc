# UI Components

This directory contains Shadcn UI components.

## Setup

Run the following to initialize Shadcn UI:

```bash
npx shadcn@latest init
```

## Components to Add

- Button
- Input
- Card
- Dialog
- Dropdown Menu
- Form
- Select
- Textarea
- Avatar
- Badge
- Calendar
- Checkbox
- Radio Group
- Switch
- Tabs
- Toast
- Tooltip
