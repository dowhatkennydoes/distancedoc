# Components

This directory contains React components for the DistanceDoc application.

## Component Structure

- `ui/` - Shadcn UI base components (Button, Input, Card, etc.)
- `VideoCall.tsx` - WebRTC video call component
- `Chat.tsx` - Real-time chat component using Firestore
- `AppointmentCalendar.tsx` - Calendar for viewing and booking appointments
- `SOAPNoteEditor.tsx` - Editor for clinical SOAP notes with AI generation
- `FileUpload.tsx` - File upload component for Cloud Storage
