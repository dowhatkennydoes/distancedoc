# Library Functions

This directory contains shared utility functions and service integrations.

## Files

- `utils.ts` - General utility functions (class names, formatting, etc.)
- `stripe.ts` - Stripe payment integration
- `webrtc.ts` - WebRTC and Xirsys TURN server configuration
